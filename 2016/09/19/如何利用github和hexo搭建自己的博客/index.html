如何利用github和hexo搭建自己的博客

Centos 安装 NodeJS：
yum -y install gcc make gcc-c++ openssl-devel wget
wget http://nodejs.org/dist/v0.10.26/node-v0.10.26.tar.gz
tar -zvxf node-v0.10.26.tar.gz
sudo make && sudo make install
node -v

安装hexo：
su（切换到root下）
npm install -g hexo

构建：
mkdir hexo
hexo init
hexo server//the same ashexo s
hexo generat//same ashexo g

配置：修改_config.yml文件
deploy:
type: git
repository: https://github.com/sophialili123/LilyBlog.git
branch: master

搭建：
hexo g
hexo d

错误：
hexo deploy //有时候会提示hexo ERROR Deployer not found:github
解决办法：

  • 安装 npm install hexo-deployer-git –save

  • 将deploy 的 type由github改为git
